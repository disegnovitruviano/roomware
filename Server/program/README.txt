RoomWare Server all-in-one
--------------------------

How to run on Windows:
1. open a Command Prompt
2. go to this directory
3. type 'bin\run.bat'


How to run on OSX:
1. open a Terminal
2. go to this directory
3. type 'bin/run'


More documentation can be found at:
http://code.google.com/p/roomware/wiki/AllInOne
