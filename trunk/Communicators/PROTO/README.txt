1. Configure your environment "vim conf/setenv"

2. Setup your environment "source conf/setenv"

3. Write your module "vim src/PROTO/Communicator.java"

4. Build your module "bin/build_module"

5. Install your module
   a. Copy the communicator jar to the modules dir of the RW Server
   b. Copy the dependecies in lib to the lib dir of the RW Server
