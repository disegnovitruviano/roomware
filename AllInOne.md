# How to run #

First unzip all the files, then:

## Windows ##
  1. open a Command Prompt
  1. go to the unzipped directory
  1. type 'bin\run.bat'


## OSX ##
  1. open a Terminal
  1. go to the unzipped directory
  1. type 'bin/run'
