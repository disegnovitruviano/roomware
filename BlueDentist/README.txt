JSR-082 implementation in progress. Device discovery works.
For PowerPC MacOSX 10.4 seen working.
Should work with recompilation on any Mac OS X > 10.2 distro & hardware.

To build:
type 'make'

Targets are made in the lib dir. You can copy these to the bluemod or RW
Server lib dir. Or use them with any other program that wants Java and
Bluetooth.
